# YANG Chat
**记得给我的项目点个⭐️Starred哦，让我更有动力维护它**

本项目于2023年3月4日启动，并发布第一个测试版本。

这是一项AI聊天程序，本程序现已接入OpenAI ChatGPT API，

您可以通过本程序帮助您写代码，写文章，以及提问。

严禁将本项目涉及违法违规以及政治领域，程序将秉持公平公正客观的原则回答您所提出的问题。

后期我们将为程序推出更多功能，敬请期待。

**本项目已经转入QT开发，发布V3.0版本**
# 使用方法
1.在release区下载msi文件并安装。

2.输入开放测试密钥73838。

3.在输入框输入您的问题。
